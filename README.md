### Projects

* ⚛️ I was part of a team that built Track-React, an open-source developer tool and Chrome DevTools extension that helps developers visualize, debug, and better understand async flows in their React apps. Learn more at [github.com/6tring/track-react](https://github.com/6tring/track-react)
* 🚀  Based on my experiences as a musician and performer, I created [Gigpromoter](https://www.gigpromoter.com) - a Node/Express application that helps new artists learn how to plan, manage, and promote live events. (Because Gigpromoter is currently in production, the repo is private - but I have created the public repo `automated-emails` to share selected files below)
* 🤖 ProofStack - A self-healing AI research agent with a producer-auditor-repairer pipeline built on Gemini API, featuring real-time SSE streaming, automated fact-checking, and session history. See repo below to view files
* 🗄️ Nexus - A data mapping management tool for mapping source values to target codes across configurable domains, with bulk operations and CSV import/export. See repo below to view files
* 🗺 Streetwise - React/Redux SPA that uses the Google Maps API to help independent artists and street teams locate venues for promotional flyer distribution. The deployed build version can be viewed at [flyering-locator-app.peterciluzzi.dev](https://flyering-locator-app.peterciluzzi.dev) or see repo below to view files
* 🎸 Fretboard - React SPA that generates visual SVG chord diagrams from a local chord database using React Context and useReducer for state management. The deployed build version can be viewed at [chord-finder-app.peterciluzzi.dev](https://chord-finder-app.peterciluzzi.dev) or see repo below to view files
